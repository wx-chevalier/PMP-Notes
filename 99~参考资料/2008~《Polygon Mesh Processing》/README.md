# Polygon Mesh Processing 笔记

## 1. 书籍基本信息

### 1.1 作者团队

- **Mario Botsch** - ETH Zurich
- **Mark Pauly** - ETH Zurich
- **Leif Kobbelt** - RWTH Aachen University
- **Pierre Alliez** - INRIA Sophia-Antipolis
- **Bruno Lévy** - INRIA Nancy

### 1.2 出版信息

- 出版年份：2008
- 出版社：AK Peters/CRC Press
- 页数：约 200 页

### 1.3 书籍背景

- 是几何处理领域第一本系统性的教材
- 汇集了当时最新的研究成果和工程实践经验
- 填补了多边形网格处理领域的教材空白

- **高级处理技术**
  - 网格修复
